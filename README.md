# chrome_extension_qrshare

##Version 1

###Commands
> **You can execute following commands in the folder of 'v1'**

1. Init The Project

    ```shell
    npm run init
    ```
2. Build For Development

    ```shell
    npm run dev
    ```
3. Build For Release

    ```shell
    npm run release
    ```
4. Get Help

    ```shell
    npm run help
    ```
